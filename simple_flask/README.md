# 簡單 Flask 應用

這是一個簡化的 Flask 應用範例，用於測試 Vercel 部署。

## 專案結構

```
simple_flask/
├── index.py           # 應用入口點
├── requirements.txt   # 依賴項
├── templates/         # HTML 模板
│   └── index.html     # 首頁模板
└── vercel.json        # Vercel 配置
```

## 部署到 Vercel

1. 將此專案推送到 GitHub 倉庫
2. 在 Vercel 上導入此倉庫
3. 確保專案設置中的框架預設為 "Other"
4. 部署!
