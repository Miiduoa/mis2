靜態版本說明

您目前正在瀏覽資訊管理系統的靜態展示版本。

靜態版本限制

由於靜態部署的特性，此版本有以下限制：

  • 無法連接至 Firebase 資料庫，因此不能新增、編輯或刪除資料
  • 無法執行伺服器端程式碼，包含 Python 功能
  • 無法執行網路爬蟲功能，電影資料為靜態示例
  • 部分功能使用 JavaScript 模擬，如登入和儀表板

模擬功能

為了展示系統的設計和界面，我們提供了以下模擬功能：

  • 模擬登入：可使用 admin@example.com / admin123 登入
  • 本地儲存：使用 localStorage 保存登入狀態
  • 靜態數據：所有頁面顯示預設的靜態數據

完整版本功能

完整版本（運行在您的本地環境）具有以下額外功能：

  • Firebase 資料庫整合，實現完整的資料管理
  • 使用者註冊、驗證和權限管理
  • 文章發布、編輯和刪除
  • 網路爬蟲獲取最新電影資訊
  • 上傳檔案和圖片功能

如何取得完整版本

完整版本的資訊管理系統需要：

  1. 從本項目的 GitHub 倉庫克隆代碼
  2. 安裝必要的 Python 套件 (requirements.txt)
  3. 設置 Firebase 專案並下載金鑰檔案
  4. 運行 Flask 應用程式 (python app.py)
